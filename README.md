## instructions:

```
git clone https://github.com/adnjoo/data-viz-ruby
cd data-viz-ruby
bundle i
yarn install
rails s
```

Tech stack: Ruby on Rails, Postgres (db), Chartkick (Highcharts), Bootstrap (styling), Heroku (deployment)
